# Security Policy

## Supported Versions

Security updates are provided for the latest version on the default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities privately to: hello@timekeepur.com

Include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

We will acknowledge receipt as quickly as possible and provide status updates during triage and remediation.
